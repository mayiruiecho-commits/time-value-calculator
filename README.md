# 这一分钟值多少钱 - 时间价值计算器

一个帮助用户快速计算每分钟薪资价值并提供实时计时赚钱功能的Web应用。

## 🎯 功能特点

### 核心功能
- **薪资计算**：输入月薪、工作天数、工作时长，自动计算每分钟和每小时薪资
- **实时计时**：启动计时器，实时显示已工作时间和已赚取金额
- **数据持久化**：自动保存设置到本地存储，下次打开无需重新输入

### 用户体验
- **移动优先**：专为移动设备优化的响应式设计
- **单页应用**：所有功能集中在一个页面，无需导航
- **实时反馈**：输入即计算，无需点击确认按钮
- **友好提示**：清晰的中文错误提示和使用说明

### 技术特性
- **性能优化**：防抖输入、计算缓存、优化渲染
- **无障碍支持**：完整的ARIA标签、键盘导航、屏幕阅读器支持
- **本地优先**：所有数据仅保存在本地，不上传服务器

## 📱 使用方法

1. **输入薪资信息**
   - 填写月薪（必填）
   - 调整每月工作天数（默认21.75天）
   - 调整每日工作时长（默认8小时）

2. **查看计算结果**
   - 每分钟薪资（主要显示）
   - 每小时薪资（辅助显示）
   - 月工作时长统计

3. **使用计时器**
   - 点击"开始"启动计时
   - 实时查看已工作时间和已赚金额
   - 可随时暂停或重置

## 🎨 设计理念

### MVP原则
- **最小功能闭环**：专注核心价值计算和计时功能
- **即时反馈**：让用户立即感受到时间的价值
- **情绪驱动**：通过实时数字增长提供正向反馈

### 用户体验
- **30秒完成计算**：简化输入流程，快速得到结果
- **反复打开**：通过计时器功能鼓励用户重复使用
- **无认知负担**：单页面设计，避免复杂导航

## 📊 性能优化

- **输入防抖**：避免频繁计算影响性能
- **计算缓存**：使用useMemo缓存计算结果
- **组件优化**：useCallback防止不必要的重渲染
- **精确计时**：使用setInterval确保计时准确性

## ♿ 无障碍支持

- **语义HTML**：正确的标题层级和表单标签
- **ARIA标签**：完整的屏幕阅读器支持
- **键盘导航**：所有功能均可通过键盘操作
- **实时更新**：使用aria-live通知状态变化
- **焦点管理**：清晰的焦点指示器

## 🔒 隐私保护

- **本地存储**：所有数据仅保存在用户设备上
- **无网络请求**：除了加载资源外，不发送任何数据
- **透明提示**：明确告知用户数据存储方式
